Installation guide (For Linux)

Install git:
```
sudo apt install git
```
Clone the git repository:
```git clone https://github.com/GITHUB-GLITCH-DEV/adminsearch```

cd to the directory:
```
cd adminsearch
```

Run the installer:
```python3 installer.py```
If you can any errors failing to install requirements ignore it

Run the file:
```python3 adminsearch.py```
